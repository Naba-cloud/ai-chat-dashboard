# AI Chat Dashboard

A modern web application built with Next.js for managing AI chat interactions, prompts, history, and analytics.

## Features

- **Dashboard**: Overview of AI chat activities
- **AI Chat**: Interactive chat interface with AI models
- **Prompts**: Manage and organize chat prompts
- **History**: View past conversations
- **Analytics**: Insights into chat usage and performance
- **Settings**: Configure AI model, temperature, tokens, and streaming options

## Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ai-chat-dashboard.git
   cd ai-chat-dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/`: Next.js app directory with pages and layouts
- `components/`: Reusable React components
  - `layout/`: Layout components (DashboardLayout, Sidebar)
  - `settings/`: Settings-related components
- `public/`: Static assets

## Usage

- Navigate through the sidebar to access different sections
- Configure AI settings in the Settings page
- Start new chats using the "New Chat" button
